# Functions:

## - output ( )

EXAMPLES:

- output with a string
```
output("hello, world!")
```
=> `hello, world!`

_____________

- output with a number
```
output(2)
```
=> `2`

_____________

- output with a constant (string)
```
word = "hi"
output(word)
``` 
=> `hi`

_________

- output with a constant (number)
```
num = 45
output(num)
```
 => `45`

______________
